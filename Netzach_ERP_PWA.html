🏛️ Pregões e Sala do Pregão

Acesse Vendas → Pregões PNCP para monitorar licitações automaticamente. Use os filtros de cidade, valor e plataforma para refinar os resultados. O sistema sincroniza a cada 30 minutos com a API pública do PNCP.

Na Sala do Pregão, selecione um pregão cadastrado para acompanhar o cronograma completo (publicação → proposta → lances → resultado), registrar seus lances, monitorar concorrentes e receber alertas sonoros durante a fase de lances.

⚠️ Os lances oficiais precisam ser dados diretamente no portal do órgão com seu certificado A3. A Sala do Pregão é para organização, estratégia e acompanhamento.
🧮 Calculadora de Proposta

Digite no campo de busca para encontrar produtos do seu cadastro automaticamente. Selecione o produto e o custo é preenchido automaticamente. Clique em ✨ Auto-calcular preços para o sistema sugerir o preço mínimo para cada item com base na margem definida. Use 📄 Exportar PDF para gerar um orçamento formal.

📦 Produtos e Fornecedores

Clique em qualquer produto para ver os preços por fornecedor diferente. Clique em qualquer fornecedor para ver todos os produtos que ele fornece com os respectivos preços. A integração de preços por fornecedor estará completa na Etapa B.

💎 AFAC

Registre cada aporte do sócio em Financeiro → AFAC. O valor aparece automaticamente no Balanço Patrimonial como passivo (obrigação da empresa com o sócio). Ao integralizar, marque como integralizado e o valor migra para Capital Social.

📄 Certidões

O sistema alerta automaticamente quando uma certidão está próxima do vencimento (30 dias = laranja, 5 dias = vermelho). Inclui certidões fiscais (CND Federal, Estadual, Municipal, FGTS, CNDT) e consultas de crédito (Serasa, SPC Empresarial).

💾 Backup

Faça backup regularmente em Gestão → Backup. O arquivo JSON contém todos os dados e pode ser restaurado a qualquer momento. A integração com Google Drive será configurada após você apresentar as credenciais.

🌙 Modo Escuro

Clique em 🌙 Escuro no menu superior para alternar. A preferência é salva automaticamente.
